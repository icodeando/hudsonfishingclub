## Hudson Fishing Club

I built this application for a fishing club in New York City. 

**Link to project:** https://hudsonfishingclub.netlify.app/

### How it's made:

**Tech used:** HTML, CSS, SASS, and JavaScript.

- Deployed to Netlify.

### Optimizations

- Use the FishWater's API https://www.fishwatch.gov/developers to complete "Search Species".
- Add a link to the main page to "Search Species" page (for mobile).
